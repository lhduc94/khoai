# KHOAI - Package for data science

**User installation**

You can install my package via `pip`
```
pip install khoai
```
or install from my source code by following:
- Clone my repo:
```git clone https://github.com/lhduc94/khoai.git```
- Go to directory
```cd khoai```
- And install my package:
```python setup.py install```
